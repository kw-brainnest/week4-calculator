Welcome to our calculator

This does simple calculations with two numbers ie. '6 / 3' or '2 + 8' etc. clicking '=' will give the rresult of these simple opeations

If an operator ('+', '-', '*', '/') is clicked after '=' has been clicked it will use the result of the last calculation as the first number for a new calculation

clicking 'C' will clear all calculations
